### How to use

1. Copy `.agent/templates/feature_passport.md` to `.agent/passports/my_feature.md`.
2. Collaborative Planning: Work with the Planner Agent to fill sections 1.1 - 1.4.
3. **Approve**: Mark the "Authorization" checkbox in the passport.
4. **Auto-Run**: Agents will automatically pick up `APPROVED` passports and move them to `REVIEW`.
